Shadow Font Styler

This tool generates the folding algorithm for making shadow text with paper when held up to the light. Each symbol in the output represents one fold unit:

  • | for perpendicular folds
  • / and \ for diagonal folds (occupying one square)
  • a blank (space) for an empty square
  • ! for a half-square fold

You can see a live preview of how this code translates into shadow text here: Shadow Font Preview .

Font: Times New Roman

⚠️ Only uppercase A–Z letters and spaces are allowed.
